Menghitung Volume Tabung dengan Fungsi
​Fungsi Program: Untuk menghitung volume tabung (V = \pi r^2 t) menggunakan fungsi berparameter.
​Cara Kerja: Program menerima input jari-jari (r) dan tinggi (t). Nilai r dan t ini diteruskan sebagai parameter ke sebuah fungsi (hitungVolumeTabung). Fungsi tersebut melakukan perhitungan dan mengembalikan hasilnya, yang kemudian dicetak.
